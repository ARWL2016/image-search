Image Search Abstraction Layer

This microservice takes in a search term and conducts an image search of the https://imgur.com/ domain. The return value is a JSON object with links to available images and the alt text property.

The API is cors-enabled for all origins.

Usage

Basic query: /api/imagesearch?search=[cats]&offset=[10]

Recent queries: /api/latest/imagesearch
